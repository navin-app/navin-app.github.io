# NAVIN Personal Carbon Tracker

🌿 Personal Carbon OS untuk Indonesia

## ✨ Features

- 🧮 Daily Carbon Calculator (GoGreener-style)
- ⚡ Activity Picker dengan 25+ predefined activities
- 📸 Bill Upload dengan auto-calculation
- 🏆 Live Leaderboard & Ranking
- 📊 Dashboard dengan summary stats
- 👤 Profile Menu dengan quick access
- 🔐 Secure login dengan Google Sheets backend

## 🚀 Quick Start

1. Buka: [NAVIN Carbon Tracker](https://yourusername.github.io/navin-carbon/)
2. Signup dengan email & password
3. Mulai tracking carbon footprint kamu!

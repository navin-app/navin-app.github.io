# 🚀 NAVIN — Deploy ke GitHub Pages (FREE)

**Waktu setup: 10 menit | Cost: FREE**

---

## STEP 1: Create GitHub Account

1. Go to: https://github.com/signup
2. Isi email, password, username
3. Verify email dari inbox
4. Login ke GitHub

**Username kamu penting! Akan jadi:**
```
https://USERNAME.github.io/navin-carbon/
```

---

## STEP 2: Create Repository

1. Login ke GitHub
2. Click **+** icon (top right)
3. Select **New repository**
4. Fill form:
   - **Repository name:** `navin-carbon`
   - **Description:** NAVIN Personal Carbon Tracker
   - **Public** (PENTING!)
   - **Initialize with README:** No (kami punya README)
5. Click **Create repository**

---

## STEP 3: Upload Files

1. Di repository, click **Add file** → **Upload files**
2. Pilih folder `navin-carbon-github/`
3. Drag & drop files:
   ```
   ✓ index.html
   ✓ design.html
   ✓ NAVIN_INDEX.html
   ✓ README.md
   ✓ NAVIN_QUICK_START.txt
   ✓ NAVIN_CALCULATOR_GUIDE.txt
   ✓ NAVIN_README.txt
   ✓ DEPLOYMENT.md
   ```
4. Click **Commit changes**

---

## STEP 4: Enable GitHub Pages

1. Go to **Settings** (tab di atas)
2. Left menu → **Pages**
3. Under "Build and deployment":
   - **Source:** Select **Deploy from a branch**
   - **Branch:** Select **main**
   - **Folder:** Select **/ (root)**
4. Click **Save**

---

## STEP 5: Wait & Get URL

1. Tunggu **1-2 menit** (GitHub building site)
2. Refresh **Settings → Pages**
3. Akan muncul:
   ```
   ✓ Your site is live at:
   https://USERNAME.github.io/navin-carbon/
   ```

## ❓ Troubleshoot

**"Site not found"?**
- Tunggu 2-3 menit lebih (GitHub Pages takes time)
- Check Settings → Pages (must show "Your site is live")

**"Cannot connect to API"?**
- Buka deployment URL di browser first untuk authorize:
- https://script.google.com/macros/s/AKfycby-fxrDhMVd0jx6m6UmNnyCIxhSDIiVB6BTW_iENDgcuZ-3svHTgsxKe9vxwwCbjU33OQ/exec
- Click Allow
- Back to app, refresh, try again

**Data not saving?**
- Check internet connection
- Look for "SYNCED" indicator (top right)
- Check browser console (F12) for errors
